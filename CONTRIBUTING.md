# Contributing

欢迎提交新的 Codex 超宽主题背景。

## 素材要求

- 推荐尺寸：`2172 × 724`，或其他接近 `3:1` 的超宽尺寸
- 左侧约 38% 保持低细节，给原生界面留出安全区
- 主要视觉焦点放在右半区
- 不包含假 UI、窗口、卡片、按钮、Logo 或水印
- 不使用未授权人物、角色、品牌或受保护素材
- 必须同时提交生成提示词或制作说明

## 文件结构

1. 将 PNG 放入 `themes/`
2. 在 `prompts/themes.json` 增加对应条目
3. 在 README 的主题预览中增加图片和说明
4. 确认 JSON 格式正确、图片能够在 GitHub 正常显示

## Pull Request

PR 请说明：

- 主题名称
- 是否深色或浅色
- 图片尺寸
- 素材权利状态
- 使用的提示词或制作方法
