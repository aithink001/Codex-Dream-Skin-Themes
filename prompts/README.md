# Theme Prompt Format

Each entry in `themes.json` contains:

- `id`: stable URL-safe identifier
- `name` / `nameZh`: English and Chinese names
- `appearance`: recommended light or dark shell
- `resolution`: downloadable source size
- `file`: relative path to the PNG theme
- `prompt`: complete reusable generation prompt
- `onlineGenerator`: a pre-attributed link to the online theme generator

The shared composition rule is intentionally repeated in every prompt so each entry can be copied independently.
